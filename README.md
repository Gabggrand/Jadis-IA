# Jadis - Discord AI Assistant 

Jadis é um projeto de inteligência artificial desenvolvido em Java, criado para funcionar como uma assistente conversacional no Discord. O objetivo do projeto foi estudar integração entre APIs, desenvolvimento back-end e criação de bots interativos.

Embora a versão do modelo de IA utilizada não seja a mais recente disponível atualmente, o projeto está totalmente funcional e demonstra a implementação prática de uma IA conversacional integrada a uma plataforma real.

---

## Funcionalidades

- Conversação com usuários no Discord
- Integração com API de inteligência artificial
- Respostas automáticas em tempo real
- Estrutura back-end desenvolvida em Java
- Sistema funcional e implantado

---

## Tecnologias Utilizadas

- Java
- Discord API / JDA
- API de Inteligência Artificial
